IRin by Helius

IRin - "IR input". Device (usb dongle) what receive signal from IR remote and send it to virtual com-port via USB (cdc-class). Build on Atmel ARM7 AT91SAM7S64 microcontroller. Use Atmel usb framework.

Main goal

I wanna to build simple device, which receive IR remote signals and send it to OS. No special software, no obsolete com-port, no difficult configuring and bind remote key. I decide to use USB communication device class (usb-cdc), because it no needs special driver. Just plug usb dongle, read VCP, now press some key on IR, in linux for example: 
$ cat /dev/ttyACM0
NEC 5916E
NEC 6916E
This asci string is remote key-code (this remote has NEC protocol).

Simple bash script example. At line 6 we read string (key code) from /dev/ttyACM0 - our device, at line 7 switching through defined keys and do action. 
 1 #!/bin/bash
 2 
 3 KEY_VOL_UP="NEC 00000"
 4 KEY_VOL_DOWN="NEC 11111"
 5 
 6 cat /dev/ttyACM0 | while read line; do
 7   case "$line" in
 8 
 9 #--- volume up action ----
10     ($KEY_VOL_UP)
11       echo "--volume up--"
12       amixer set Master 2+ > /dev/null
13     ;;
14 
15 #--- volume down action ---
16     ($KEY_VOL_DOWN)
17       echo "--volume down--"
18       amixer set Master 2+ > /dev/null
19     ;;
20 
21 #--- template to you action ---
22   # (YOUR KEY)
23   #   echo "--you action--"
24   # ;;
25 
26 #--- default action, if key not matched ---
27     (*)
28       echo "--unknown key $line"
29     ;;
30   esac
31 done

Schematic

Schemaitic is very simple, it contains
  • AT91SAM7S64,
  • 18.342 MHz crystal,
  • 3.3V low drop line regulator,
  • Atmel AT24 eeprom,
  • 2color led for indication,
  • TSSOP1736 IR receiver,
  • few passive components.

Firmware

You can download intell hex file and flash it to chip.

How to flash chip

I use sam_i_am, thanks to Andrew Sterian for cool utility.

Build

You need crosscompiler for build project, i use summon-arm-toolchain scripts for build gcc-arm toolchain.
Now you can tune project via Makefile and build it: 
$cd src/usb-device-cdc-serial-project/
$make
